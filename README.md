## 🚀 One-Click Quick Start

### 1. Installation
```bash
git clone https://github.com/Maliot100X/BinanceClawBot.git
cd BinanceClawBot
pip install -r requirements.txt
cd dashboard && npm install && cd ..
```

### 2. Configure Brain
```bash
cp .env.example .env
# Fill in BINANCE_API_KEY, BINANCE_SECRET_KEY, and TELEGRAM_BOT_TOKEN
```

### 3. Professional One-Click Launch
The new `start_all.py` script automatically verifies dependencies (numpy, pandas, etc.) and launches the API, Telegram Bot, and Dashboard concurrently.
```bash
py start_all.py
```

## 🔑 Professional OAuth (PKCE)
KaiNova uses a high-security PKCE flow for OpenAI, Gemini, and Antigravity. No plain API keys are stored.

### CLI Authentication
```bash
py codex.py login --provider openai
```

### Web Authentication
Visit `http://localhost:3000` and click **Connect Brain 🧠** in the top navbar. You can preview the dashboard even before connecting!
